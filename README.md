# SplitReader

An Android e-book reader with real-time translation. Open an EPUB or FB2 file and read the original text side-by-side with a live translation.

## Features

- EPUB and FB2 format support
- Split-pane layout: original text + translation
- Auto language detection (ML Kit)
- 12 target languages: EN, RU, DE, FR, ES, IT, ZH, JA, PT, AR, KO, TR
- Translation cache — no repeated API calls
- Reading progress saved per book

## Tech Stack

Kotlin · Jetpack (Hilt, Room, Navigation) · ML Kit Translate · Coroutines + Flow

## Requirements

- Android 8.0+ (API 26)

## Status

Active development — more features coming.
